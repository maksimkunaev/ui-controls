# ui-controls

## **Usage**

```
const wrapper = document.getElementById("app");

const object = {
    name: 'John',
    isAdmin: false,
    age: 28,
    note: 'Working a driver',
}

const controls = new Controls(wrapper, object)

controls.add('name')
controls.add('age')
controls.add('isAdmin')
```
